\section*{Цель работы}
\addcontentsline{toc}{section}{Цель работы}
Изучение принципов объектно ориентированного программирования на языке C++.
\section*{Задание}
\addcontentsline{toc}{section}{ Задание }

\begin{enumerate}
	\item Выбрать и согласовать с преподавателем предметную область.
	\item Создать иерархию классов, состоящую не менее чем из одного родительского и двух дочерних классов.
	\item В каждом классе определить не менее двух членов данных, не менее двух собственных, а для дочерних — не менее двух унаследованных и двух переопределенных (перекрытых) член-функций.
	\item Разработать приложение, демонстрирующее принципы инкапсуляции, наследования и полиморфизма.
\end{enumerate}
* Работу с объектами осуществлять только через переменные родительского типа. \\
* Каждый класс в отдельном файле. \\
* Отчет должен содержать описание предметной области, схему структуры иерархии классов (можно использовать диаграмму классов), описание методов, особенности реализации.

\subsection*{Разработка демонстрационного приложения}
\addcontentsline{toc}{subsection}{Разработка демонстрационного приложения}
Для демонстрации созданной структуры разработано конскольное приложение.

\textbf{Демонстрация инкапсуляции:}

Данные классов (revenue, purpose) скрыты в секции private. Доступ к ним и изменение состояния объектов возможны только через публичные методы.

\textbf{Демонстрация наследования:}

Объекты Commersorg и Noncomersorg содержат все поля и методы класса Organization.При вызове метода ARGANIZATII[2]->expandBusiness(), который не определен в базовом классе, но успешно вызывается у объекта дочерней коммерческой организации благодаря наследованию и полиморфизму.

\textbf{Демонстрация полиморфизма:}

Все объекты хранятся в векторе указателей на базовый класс std::vector<Organization*>. При вызове виртуальных методов (payTaxes(), report()) программа автоматически выполняет версию метода, соответствующую реальному классу объекта.

В консоль будет выведен налог для коммерческой организации, отчет для коммерческой организации, и вызван метод expandBusiness(), который увеличит количество подписчиков.
